

# Laravel Widgets

[![Software License](https://img.shields.io/badge/license-MIT-brightgreen.svg?style=flat-square)](LICENSE.md)

Widgets integration package for the laravel framework.

## Install

Via Composer

``` bash
$ composer require humweb/laravel-widgets
```
